Course Content

Introduction to HTML

This course covers the basics of HTML (Hypertext Markup Language), the standard markup language for documents designed to be displayed in a web browser.

Topics covered include:

  • HTML document structure
  • HTML elements and tags
  • Text formatting
  • Lists and tables
  • Links and images

CSS Fundamentals

This course covers the fundamentals of CSS (Cascading Style Sheets), the language used to style the presentation of a document written in HTML.

Topics covered include:

  • CSS syntax and selectors
  • Styling text and backgrounds
  • Box model and layout
  • CSS properties and values
  • CSS positioning

JavaScript Basics

This course covers the basics of JavaScript, the programming language that enables interactive web pages and web applications.

Topics covered include:

  • Variables and data types
  • Operators and expressions
  • Control flow and loops
  • Functions and scope
  • DOM manipulation
